# Retro-Calc — Demo de calculadora retro (prototipo)

## Descripción / Description
Proyecto: prototipo de calculadora visual estilo “retro” implementado en una sola página (`index.html`). Interfaz responsive y accesible con estética retro y teclado en pantalla.  
Project: Retro-style calculator single-page prototype (`index.html`). Responsive, accessible UI with a retro look and on-screen keypad.

## Estado / Status
- Prototipo funcional (v1): operaciones básicas (+, −, ×, ÷), paréntesis, decimales.  
- Entrada por teclado y botones; `Enter` = igual, `Backspace` = borrar.  
- Evaluación segura mediante Shunting-yard → RPN (no se usa `eval`).  
- Manejo básico de errores (muestra “Error” en expresiones inválidas).  
- Prototype functional (v1): basic ops, parentheses, decimals. Keyboard + on-screen input. Safe evaluation via Shunting-yard (no eval). Basic error handling.

## Cómo probar / How to run
1. Abrir `index.html` en un navegador moderno (doble clic o arrastrar/abrir).  
2. Usar teclado o hacer clic en los botones; probar operaciones, paréntesis y decimales.  
1. Open `index.html` in a modern browser (double-click or open file).  
2. Use keyboard or click buttons; test operations, parentheses and decimals.
